# Release Notes — v0.3.14

**Date:** 2025-09-05
**Repository:** scholar-rag-agent

## Highlights

This release focuses on stability and developer-experience improvements
to the `agent` core.

## Changes

### Added
- `grounding` module: added configurable retry strategy with jitter
- New `--verbose` flag for detailed retrieval trace output
- Unit tests for arXiv boundary conditions

### Fixed
- Off-by-one in pagination logic when result set is empty
- Memory leak in long-running grounding worker loop
- Incorrect timezone handling in audit timestamps

### Improved
- Reduced average retrieval latency by ~12% via connection pooling
- Cleaner error messages for arXiv validation failures

## Upgrade Notes

No breaking changes. Drop-in upgrade from the previous version.

```bash
pip install --upgrade scholar-rag-agent
```
