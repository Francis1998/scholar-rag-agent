# Release Notes — v0.4.8

**Date:** 2024-03-14
**Repository:** scholar-rag-agent

## Highlights

This release focuses on stability and developer-experience improvements
to the `agent` core.

## Changes

### Added
- `ingestion` module: added configurable retry strategy with jitter
- New `--verbose` flag for detailed graph trace output
- Unit tests for retrieval boundary conditions

### Fixed
- Off-by-one in pagination logic when result set is empty
- Memory leak in long-running ingestion worker loop
- Incorrect timezone handling in audit timestamps

### Improved
- Reduced average graph latency by ~12% via connection pooling
- Cleaner error messages for retrieval validation failures

## Upgrade Notes

No breaking changes. Drop-in upgrade from the previous version.

```bash
pip install --upgrade scholar-rag-agent
```
